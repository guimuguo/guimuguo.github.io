  • Configure your COMPUTER SYSTEM and ACCESS:

    1. Confirm access to your Rowan Network Username & Password (different than your student ID# - usually your last name & first initial)
    2. Review access to your student information and course registration in Rowan Banner Self-Service (using your 9-digit Banner ID & PIN)
    3. Find your H: drive for Network File Storage
    4. Connect to the University's secure wireless network RowanSecure after you:
      • Secure your personal computer with anti-virus software
      • Complete Rowan ClearPass Registration on your laptop (if you want to use it for Rowan network access on campus)
    5. Utilize Rowan Virtual Desktop or mount your home directory with Rowan ExpanDrive (if you require remote access to your H: drive)
    6. Install Office 365 with the full Office suite on up to 5 personal devices for free
    7. Access Rowan-owned software from any device using RowanCloud-Citrix on or off campus (if necessary)
    8. Create a folder called IOOP or other course name on your H: drive to store all your class projects and work (using File Manager)
    9. (IOOP) Install the platform-specific BlueJ Application and JDK for your system (MUST use at least BlueJ version 4.2.2 and Java 8)
    10. (IOOP) Download the BlueJ projects from the web
    11. (OPTIONAL) Make your own web presence by:
      • Finding the public.www directory under your H: drive ... If not there, create a new directory named public.www
      • Opening a text editor such as Notepad
      • Writing your name in the text file ... Later, look at updating this file with HTML
      • Saving the text file with the exact name index.html
      • Accessing your web page from a browser using the URL: users.rowan.edu/~yourNetworkUsername
